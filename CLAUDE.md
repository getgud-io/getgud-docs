# GetGud Documentation

Platform and SDK integration documentation.

## Structure
- `get-started.md` — quickstart guide
- `sdk-commands.md` — SDK API reference
- `1-Integrations/` — engine-specific guides (Unity, Unreal, C++ build instructions)
- `2-Platform/` — dashboard and platform usage guides
- `3-Extra/` — additional integration guides (C, C#, Python)

## Rules
- Keep language simple and direct — target audience is game developers
- All code examples must be tested and working
- Reference actual SDK API from `cpp-getgud-sdk-dev/include/GetgudSDK.h`
